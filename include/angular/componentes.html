1. DATABINDING

DATABINDING

INTERPOLACAO

É a forma mais facil de exibir um dado. O dados é representado no template por dois pares de chaves. 
<label>Nome:{{nome}}</label>

PROPERTY BINDING

Define o valor de uma propriedade do elemento HTML, é o conceito de one-wayque o valor só vai para e NAO VOLTA! 
<img [src]="minha-url">

EVENT BINDING

Realiza a chamada de uma metodo com base no evento do elemento HTML. 
<button (click)="clicContador()">CLIQUE AQUI<button>

TWO-WAY BINDING

Exibe e atualiza um dado nas duas direcoes(template/componente) 
<input type="text" [(ngModel)]="evento.nome">
<label>Nome:{{evento.nope}}</label>
