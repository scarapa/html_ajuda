BOOTSTRAP

No angular também é possivel inserir o bootstrap.

Instalando

$ npm install bootstrap@4.1.1

Carregando no APP

Esta indicação é feita em angular.json, que possui uma chave denominada build, dentro do qual está styles. Alteraremos e salvaremos seu conteúdo para incluir o caminho completo do Bootstrap na pasta "node_modules": 
"styles": [
    "src/styles.css",
    "./node_modules/bootstrap/dist/css/bootstrap.min.css"
],
"scripts": []

CommonModule

Sempre que criar um componente , como uma boa pratica SEMPRE incluir o CommonModule 
import { NgModule } from '@angular/core';
import { ReactiveFormsModule } from '@angular/forms';
import { CommonModule } from '@angular/common';

@NgModule({
    declarations: [SignInComponent] ,
    imports: [ 
        CommonModule ,
        ReactiveFormsModule ,
    ],
})

export class HomeModule {

}
