REPLACE

O MySQL REPLACE() substitui todas as ocorrências de uma substring dentro de uma string. 
REPLACE(str, find_string, replace_with)
exemplo: 
UPDATE posts SET post = REPLACE(post,'microsof','linux') WHERE post LIKE '%microsoft%'
