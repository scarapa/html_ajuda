CRIAR TRIGGER

SQLITE

Para criar um novo gatilho no SQLite, você usa a instrução CREATE TRIGGER , mas só pode dar uma UNICA ACAO INSERT|UPDATE|DELETE, utiliza-se da seguinte maneira: 
CREATE TRIGGER [IF NOT EXISTS] trigger_name 
    [BEFORE|AFTER|INSTEAD OF] [INSERT|UPDATE|DELETE] 
    ON table_name
    [WHEN condition]
 BEGIN
  statements;
 END;
INSERT 
CREATE TRIGGER coluna_update_at_action_insert AFTER INSERT ON producao_leites
BEGIN
    UPDATE producao_leites SET updated_at =  datetime('now','localtime') WHERE id = NEW.id;
END;
UPDATE 
CREATE TRIGGER coluna_update_at_action_update AFTER UPDATE ON producao_leites
BEGIN
    UPDATE producao_leites SET updated_at =  datetime('now','localtime') WHERE id = NEW.id;
END;
