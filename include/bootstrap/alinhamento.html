Alinhamento

Realinhe facilmente o texto aos componentes com as classes de alinhamento de texto 
<p class="text-left">Left aligned text on all viewport sizes.</p>
<p class="text-center">Center aligned text on all viewport sizes.</p>
<p class="text-right">Right aligned text on all viewport sizes.</p>

<p class="text-sm-left">Left aligned text on viewports sized SM (small) or wider.</p>
<p class="text-md-left">Left aligned text on viewports sized MD (medium) or wider.</p>
<p class="text-lg-left">Left aligned text on viewports sized LG (large) or wider.</p>
<p class="text-xl-left">Left aligned text on viewports sized XL (extra-large) or wider.</p>

Text wrapping and overflow

<div class="badge badge-primary text-wrap" style="width: 6rem;">
  This text should wrap.
</div>


<div class="text-nowrap bd-highlight" style="width: 8rem;">
  This text should overflow the parent.
</div>

Para um conteúdo mais longo, você pode adicionar uma classe .text-truncate para truncar o texto com reticências. Requer exibição: bloco embutido ou exibição: bloco. 
<!-- Block level -->
<div class="row">
  <div class="col-2 text-truncate">
    Praeterea iter est quasdam res quas ex communi.
  </div>
</div>

<!-- Inline level -->
<span class="d-inline-block text-truncate" style="max-width: 150px;">
  Praeterea iter est quasdam res quas ex communi.
</span>

Word break

Evite que longas sequências de texto quebrem o layout de seus componentes usando .text-break para definir overflow-wrap: break-word (e word-break: break-word para compatibilidade com o IE e Edge). 
<p class="text-break">mmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmmm</p>

Flex

Criação de contêiner flexbox e transformar elementos filhos diretos em itens flex. Os contêineres e itens flexíveis podem ser modificados ainda mais com propriedades flexíveis adicionais.

Direction

Defina a direção dos itens flexíveis em um contêiner flexível com os utilitários de direção. Na maioria dos casos, você pode omitir a classe horizontal aqui, pois o padrão do navegador é linha. No entanto, você pode encontrar situações em que precisou definir explicitamente esse valor (como layouts responsivos).

Use .flex-row para definir uma direção horizontal (o padrão do navegador) ou .flex-row-reverse para iniciar a direção horizontal do lado oposto. 
<div class="d-flex flex-row bd-highlight mb-3">
  <div class="p-2 bd-highlight">Flex item 1</div>
  <div class="p-2 bd-highlight">Flex item 2</div>
  <div class="p-2 bd-highlight">Flex item 3</div>
</div>
<div class="d-flex flex-row-reverse bd-highlight">
  <div class="p-2 bd-highlight">Flex item 1</div>
  <div class="p-2 bd-highlight">Flex item 2</div>
  <div class="p-2 bd-highlight">Flex item 3</div>
</div>
Use .flex-column to set a vertical direction, or .flex-column-reverse to start the vertical direction from the opposite side. 
<div class="d-flex flex-column bd-highlight mb-3">
  <div class="p-2 bd-highlight">Flex item 1</div>
  <div class="p-2 bd-highlight">Flex item 2</div>
  <div class="p-2 bd-highlight">Flex item 3</div>
</div>
<div class="d-flex flex-column-reverse bd-highlight">
  <div class="p-2 bd-highlight">Flex item 1</div>
  <div class="p-2 bd-highlight">Flex item 2</div>
  <div class="p-2 bd-highlight">Flex item 3</div>
</div>

Align items

Use os utilitários de alinhamento de itens nos contêineres do flexbox para alterar o alinhamento dos itens flex no eixo cruzado (o eixo y a ser iniciado, o eixo x se a direção da flexão: coluna). Escolha entre início, fim, centro, linha de base ou extensão (padrão do navegador). 
<div class="d-flex align-items-start">...</div>
<div class="d-flex align-items-end">...</div>
<div class="d-flex align-items-center">...</div>
<div class="d-flex align-items-baseline">...</div>
<div class="d-flex align-items-stretch">...</div>
