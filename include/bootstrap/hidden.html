Para um desenvolvimento mais rápido para dispositivos móveis, use classes de exibição responsivas para mostrar e ocultar elementos por dispositivo. Evite criar versões totalmente diferentes do mesmo site; oculte o elemento responsivamente para cada tamanho de tela.

Para ocultar elementos, basta usar a classe .d-none ou uma das classes .d- {sm, md, lg, xl} -one para qualquer variação de tela responsiva.

Para mostrar um elemento apenas em um determinado intervalo de tamanhos de tela, você pode combinar uma classe .d -* -none com uma classe .d-*-*, por exemplo .d-none .d-none .d-md-block .d-xl-none ocultará o elemento para todos os tamanhos de tela, exceto em dispositivos médios e grandes.
