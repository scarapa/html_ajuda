ENDPOINT

Como o Bootstrap foi desenvolvido para ser móvel primeiro, usamos várias consultas de mídia para criar pontos de interrupção sensíveis para nossos layouts e interfaces. Esses pontos de interrupção baseiam-se principalmente nas larguras mínimas da janela de visualização e permitem escalar elementos à medida que a janela de visualização muda.

O Bootstrap usa principalmente os seguintes intervalos de consulta de mídia - ou pontos de interrupção - em nossos arquivos Sass de origem para nosso layout, sistema de grade e componentes.
