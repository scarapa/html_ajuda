A pseudo-classe CSS de negação, :not(X), é uma notação funcional que recebe um seletor simples X como argumento. Ela seleciona um elemento que não é representado por seu argumento. X não pode conter outro seletor de negação.
exemplo: Para retirarmos a borda da ultima linha, tinhamos que resetar as propriedades do ultimo item, caso do exemplo abaixo , o item 3
  • item 1
  • item 2
  • item 3
CSS
#exemplo1 li { border-bottom: 2px solid red}
#exemplo1 li:last-child { border-bottom: 0; }
Algo mais elegante seria mais ou menos assim ...
CSS
#exemplo1 li:not(:last-child) { border-bottom: 2px solid red}
