Comandos Básicos - Terminal

Help $ sass -h
Converter arquivo scss para sass

sass convert nome arquivo scss já existente e o nome do arquivo sass que irá ser criado

$ sass-convert Example.scss Example.sass
Converter arquivo sass para scss

sass-convert nome arquivo scss já existente e o nome do arquivo sass que irá ser criado

$ sass-convert Example.sass Example.scss
Converter arquivo scss para css

sass nome arquivo scss já existente, dois pontos com o nome do arquivo css que irá ser criado

$ sass Example.scss:Example.css
Observar arquivo com scss - (conversão automática do arquivo scss para css)

sass --watch com o nome arquivo scss já existente que irá ser observado, dois pontos com o nome do arquivo css

$ sass --watch Example.scss:Example.css
Observar diretório com arquivos scss - (conversão automática do diretório com arquivos scss para css)

sass --watch com o nome do diretório de arquivos scss já existente que irá ser observado, dois pontos com o nome de um novo diretório, ou de um já existente, onde serão gerados (ou modificados) os arquivos css

$ sass --watch Directory:CSS
Converter arquivo scss para css com compressão de estilo

sass nome arquivo scss já existente, dois pontos com o nome do arquivo css que irá ser criado --style com o tipo de compressão desejada

  • compact - identificadores em um linha
  • compress - tudo em um linha e sem espaços em branco
  • expanded - formatado padrão
$ sass Example.scss:Example.css --style compact
$ sass Example.scss:Example.css --style compress
$ sass Example.scss:Example.css --style expanded
Converter arquivo scss para css sem gerar arquivo map

sass nome arquivo scss já existente, dois pontos com o nome do arquivo css que irá ser criado mais --sourcemap=none

$ sass Example.scss:Example.css --sourcemap=none

Também pode ser usado com diretórios ao invés de arquivos
