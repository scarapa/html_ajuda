Inserir coluna com numeração automática

CSS
.tableNumeracaoAutomatica tr:not(.fble_htr) {
    counter-increment: rowNumber;
}

.tableNumeracaoAutomatica tr:not(.fble_htr) td:first-child::before {
    content: counter(rowNumber);
    min-width: 1em;
    margin-right: 0.5em;
}
