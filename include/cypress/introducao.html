Cypress é uma ferramenta de teste, criada para a web moderna. Aborda os principais pontos problemáticos que desenvolvedores e engenheiros de QA enfrentam ao testar aplicativos modernos.

Simplificamos:
  • Configurar testes
  • Escrever testes
  • Executar testes
  • Testes de depuração
Cypress é mais frequentemente comparado ao selênio; no entanto Cypress é fundamentalmente e arquitetonicamente diferente. Cypress não é restringido pelas mesmas restrições que o selênio.
Isso permite que você escreva testes mais rápidos, fáceis e confiáveis.Cypress permite que você escreva todos os tipos de testes:
  • Testes de ponta a ponta
  • Testes de integração
  • Testes unitários


Cypress pode testar tudo o que é executado em um navegador.
