.GET()

é utilizado para buscar algum elemento na tela, funciona de forma semelhando a seleção de elementos do jQuery e ao comando document.querySelector do javascript padrão. Dentro do pode estar um id, classe, elemento, qualquer coisa que identifique o que deve ser selecionado; 
cy.get('seletor')

.VISIT('url')

redireciona para a url incluída; 
cy.visit(‘url’)

.LOGIN()

comando criado no arquivo commands.js para facilitar o login dentro dos sistemas da casa; 
cy.login(‘’)

.WAIT()

faz com o teste “espere” por determinado tempo em milissegundos, caso alguma rotina precise esperar alguma ação acontecer para seguir a execução normal do teste; Assim como o javascript padrão, o cypress permite encadear comando, como os exemplos abaixo: 
cy.wait(‘tempo’)

.CLICK()

busca o elemento e efetua o click; 
cy.get('seletor').click()

.CHECK()

busca o elemento (radio button ou checkbox) e faz a marcação; 
cy.get('seletor').check()

.SELECT()

Utilizado para selecionar um valor no elemento SELECT 
cy.get('seletor').select('VALOR');

.TYPE()

Utilizado para escrever dentro do input tipo texto 
cy.get('seletor').type("Fulano da Silva");
