Parametros POSICIONAIS

class Cliente{
    String nome;
    String cpf;

    Cliente(String nome, String cpf){
        this.nome = nome;
        this.cpf = cpf;
    }
}
ou da forma abaixo: 
class Cliente{
    String nome;
    String cpf;

    Cliente(this.nome , this.cpf);
}
Construindo .... 
main(){
    var c1 = Cliente('Joao','123.456.789-00');
    print("${c1.nome} ${c1.cpf}")
}

Parametros NOMEADOS

Basta dispor os atributos entre chaves. 
class Cliente{
    String nome;
    String cpf;

    Cliente({this.nome , this.cpf});
}
Isso permite construir objeto da sua forma.
Construindo .... 
main(){
    var c1 = Cliente(
        cpf: '123.456.789-00' , 
        nome:'Joao'
    );
    print("${c1.nome} ${c1.cpf}")
}
Fazendo dessa forma, não é INCOMUM quando estanciar o objeto, separar os atributos por linhas.

GETTERS AND SETTERS

class Produto{
    int codigo;
    String nome;
    double preco;
    double desconto;

    Produto({this.codigo , this.nome , this.preco , this.desconto = 0});

    // PROPRIEDADE QUE CALCULA O DESCONTO
    double get precoComDesconto{
        return ( 1 - desconto) * preco;
    }

    double getPrecoComDesconto(){
        return ( 1 - desconto) * preco;
    }
}

main(){
    var p1 = Produto(
        codigo:1,
        desconto: 0.13,
        nome:'Caneta',
        preco: 13.50
    );

    print( p1.precoComDesconto );
    // 11.745

    print( p1.getPrecoComDesconto() );
    // 11.745    
}
