Metodos importantes

forEach()

Executa uma função em cada elemento da lista 
var fruits = [‘banana’, ‘pineapple’, ‘watermelon’];
fruits.forEach((fruit) => print(fruit)); // => banana pineapple watermelon

map()

Produz uma nova lista após transformar cada elemento em uma determinada lista 
var mappedFruits = fruits.map((fruit) => ‘I love $fruit’).toList();
print(mappedFruits); // => ['I love banana', ‘I love pineapple’, ‘I love watermelon’]

contains()

Verifica para confirmar se o elemento fornecido está na lista 
var numbers = [1, 3, 2, 5, 4];
print(numbers.contains(2)); // => true

sort()

Ordene os elementos com base na função de ordenação fornecida 
numbers.sort((num1, num2) => num1 - num2); // => [1, 2, 3, 4, 5]

reduce(), fold()

Compacta os elementos em um único valor, usando a função fornecida 
var sum = numbers.reduce((curr, next) => curr + next);
print(sum); // => 15const initialValue = 10;
var sum2 = numbers.fold(initialValue, (curr, next) => curr + next);
print(sum2); // => 25

every()

Confirma que cada elemento satisfaz o teste 
List<Map<String, dynamic>> users = [
{ “name”: ‘John’, “age”: 18 },
{ “name”: ‘Jane’, “age”: 21 },
{ “name”: ‘Mary’, “age”: 23 },
];var is18AndOver = users.every((user) => user[“age”] >= 18);
print(is18AndOver); // => true

var hasNamesWithJ = users.every((user) => user[“name”].startsWith('J'));
print(hasNamesWithJ); // => false

where(), firstWhere(), singleWhere()

Retorna uma coleção de elementos que satisfazem um teste 
// Consulte o nº 6 para obter a lista de usuários
var over21s = users.where((user) => user[“age”] > 21);
print(over21s.length); // => 1var nameJ = users.firstWhere((user) => user[“name”].startsWith(‘J’), orElse: () => null);
print(nameJ); // => {name: John, age: 18}var under18s = users.singleWhere((user) => user[“age”] < 18, orElse: () => null);
print(under18s); // => null
firstWhere() retorna a primeira correspondência na lista, enquanto singleWhere() retorna a primeira correspondência, desde que haja exatamente uma correspondência.

take(), skip()

Retorna uma coleção enquanto inclui ou pula elementos 
var fiboNumbers = [1, 2, 3, 5, 8, 13, 21];
print(fiboNumbers.take(3).toList()); // => [1, 2, 3]
print(fiboNumbers.skip(5).toList()); // => [13, 21]
print(fiboNumbers.take(3).skip(2).take(1).toList()); // => [3]

List.from()

Cria uma nova lista da coleção fornecida 
var clonedFiboNumbers = List.from(fiboNumbers);
print(‘Cloned list: $clonedFiboNumbers’);
A partir do Dart 2.0, a new palavra-chave é opcional ao instanciar objetos.

expand()

Expande cada elemento em zero ou mais elementos 
var pairs = [[1, 2], [3, 4]];
var flattened = pairs.expand((pair) => pair).toList();
print(‘Flattened result: $flattened’); // => [1, 2, 3, 4]var input = [1, 2, 3];
var duplicated = input.expand((i) => [i, i]).toList();
print(duplicated); // => [1, 1, 2, 2, 3, 3]

Metodos mais simples

A lista abaixo sao os metodos mais simples , mas nao menos importantes, pois sao muito utilizados.

add

adiciona um novo elemento na lista fornecida.

length

retorna o número total de elementos em determinada lista.

isEmpty

retorna booleano se a lista fornecida estiver vazia ou não.

isNotEmpty

verifica e retorna booleano se a lista fornecida tiver elementos.

first

retorna o primeiro elemento da lista fornecida.

last

retorna o último elemento da lista fornecida.

clear

limpa a lista fornecida.

Juntar Listas

Dart <2

var newList = new List.from(list1)..addAll(list2);
var newList = [list1, list2, list3].expand((x) => x).toList()

Dart <2

var newList = list1 + list2 + list3;

Dart >2.3

var newList = [...list1, ...list2, ...list3];
