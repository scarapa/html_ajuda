For In

O loop for ... in é usado para percorrer as propriedades de um objeto. A seguir está a sintaxe do loop ‘for… in’. 
for (variablename in object){  
    statement or block to execute  
}
Em cada iteração, uma propriedade do objeto é atribuída ao nome da variável e esse loop continua até que todas as propriedades do objeto sejam exauridas. 
void main() { 
    var obj = [12,13,14]; 
    
    for (var prop in obj) { 
       print(prop); 
    } 
 }

E para utilizar o index caso haja uma necessidade existem algumas maneiras

Map Entries

Converta a lista em um mapa e, em seguida, mapeie as entradas que contêm os pares chave/valor. Cada chave no mapa é o índice da lista original. 

  ...listagem.asMap().entries.map((entry) {
    int idx = entry.key;
    String val = entry.value;    
    return Text(val);
  }),

Gerar uma lista

Se você estiver percorrendo várias listas com comprimento fixo, pode ser mais eficiente gerar uma única lista uma vez. Por exemplo, criamos uma lista variada com o mesmo tamanho da original 
  final List fixedList = Iterable.generate(listagem.length).toList();
  fixedList.map((idx) {
    String val = myList[idx];
    return Text(val);
  }

Pesquise pelo objeto na lista

Você pode acessar o índice de um valor específico procurando por ele com List.indexOf, que retorna o índice da primeira correspondência. Essa abordagem é mais previsível quando todos os valores são exclusivos. Um conjunto pode garantir exclusividade em toda a lista. 
  ...listagem.map((nome){
    int index = listagem.indexOf(nome);
    return Text(nome);
  }),
