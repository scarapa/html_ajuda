BASICO

1 - BAIXAR UMA IMAGEM

$ docker container run -p 8080:80 IMAGEM_NOME
Startando CONTAINER 
$ docker start CONTAINER_NOME

2 - ACESSANDO CONTAINER

Pode ser feito de duas formas, ou passando o comando ou entrando DENTRO do container

2.1 - Passando o comando e recebendo o retorno

$ docker container exec nginx uname -or
4.9.0-deepin4-amd64 GNU/Linux

2.2 - Entrando no CONTAINER via terminal

$ docker exec -i -t CONTAINER_NOME /bin/bash
root@240d03e9502d:/usr/local/apache2#

Para sair basta digitar: exit
