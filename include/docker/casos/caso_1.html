Caso 01

Primeiro desafio será criar um ambiente de desenvolvimento com as seguintes ferramentas
  1. Apache
  2. PHP 7.0+
  3. MYSQL 5+
  4. PHPMyAdmin

1. Baixar as imagens

Apache + php7 $ docker pull nimmis/apache-php7
MYSQL $ docker pull mysql
PHPMYADMIN $ docker pull phpmyadmin/phpmyadmin

Listar as imagens baixadas

$ docker image ls

2 Levantando os Containers

Startando os containers 
$ docker run --name server nimmis/apache-php7
$ docker run --name
OBS: Pasta onde ficam salvas os containes /var/lib/docker/containers/ Levantando APACHE Na pagina da documentação diz se você for rodar o httpd de um rodar isolado(Without a Dockerfile) é preciso iniciar dessa forma: docker run -dit --name apache-web -v "$PWD":/usr/local/apache2/htdocs/ httpd
