IMAGES

DOWNLOAD

$ docker pull [IMAGEM_NOME]
EXEMPLO: 
$ docker pull beevelop/ionic:latest

ACESSAR

$ docker run -it [IMAGEM_NOME] /bin/sh
EXEMPLO: 
$ docker run -it beevelop/ionic bash

UTILIZANDO

FROM beevelop/ionic:latest

CONTAINER

START

$ docker container start [CONTAINER_NOME]

STOP

$ docker container stop [CONTAINER_NOME]

ACESSAR

$ docker container exec -ti [CONTAINER_NOME] [COMANDO]
EXEMPLO: 
$ docker container exec -ti [CONTAINER_NOME] /bin/sh

LISTAR

Lista todos containers em execucao 
$ docker container ps
Lista todos containers criados 
$ docker container ps -a

DELETAR

$ docker container rm [CONTAINER_ID]

OBS[2]: modo interativo: o container esta sendo executado ;modo daemon: o container esta sendo executado em background
