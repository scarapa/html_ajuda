Removendo versões antigas

$ sudo apt-get remove docker docker-engine docker.io
As imagens ficam preservadas em: /var/lib/docker/

Atualizando o sistema e instalar alguns pacoetes

$ sudo apt-get update
$ sudo apt-get install \ apt-transport-https \ ca-certificates \ curl \ software-properties-common

Instalar o repósitorio oficial do DOCKER

$ curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -
Use o seguinte comando abaixo para configurar o repositório estável. Você sempre precisa do repositório estável, mesmo se você quiser instalar versões da borda ou repositórios de teste também. Para adicionar a borda ou repositório de teste, adicione a borda da palavra ou teste (ou ambos) após a palavra estável nos comandos abaixo. 
$ sudo add-apt-repository \
   "deb [arch=amd64] https://download.docker.com/linux/ubuntu \
   $(lsb_release -cs) \
   stable"

Instalando DOCKER CE

$ sudo apt-get update
$ sudo apt-get install docker-ce
