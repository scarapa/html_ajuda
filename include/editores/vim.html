Acessar o VIM $ vim

Dentro do VIM

Digitar i para começar a escrevar


Apertar ESC para sair do modo de edição

Fora do modo de edição

Digitar "i": vai para o modo de edição

Digitar "ESC": sai do modo de edição

Digitar ":w": salva o arquivo

Digitar ":wq": salva o arquivo e sai do vim

Digitar ":q": sai do vim

Digitar ":q!": força saída do vim

Digitar "0": vai para o início da linha

Digitar "$": vai para o fim da linha

Digitar "u": desfazer última ação

Digitar "x": deletar caracter onde o cursor se encontra

Digitar "dd": deletar linha

Digitar "p": colar texto deletado

Digitar "Ctrl+F": vai uma tela para frente

Digitar "Ctrl+B": vai uma tela para trás

Digitar "G": vai para a última linha

Digitar "1G": vai para a primeira linha

Digitar "/": procura por
