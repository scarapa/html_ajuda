Para reproduzir Ativos Locais, você deve usar a classe AudioCache.

O Flutter não fornece uma maneira fácil de reproduzir áudio em seus ativos, mas essa classe ajuda muito. Na verdade, ele copia o ativo para uma pasta temporária no dispositivo, onde é reproduzido como um arquivo local.

Funciona como um cache porque mantém o controle dos arquivos copiados para que você possa reproduzi-los sem demora; você também pode pré-carregar arquivos.

Cada instância tem seu próprio cache independente, portanto, se você quiser otimizar seus tempos de carregamento, crie um singleton.

Para reproduzir um áudio, basta executar:

Instalação

Insira no arquivo pubspec.yaml 
dependencies:
    flutter:
      sdk: flutter
    audioplayers: ^0.10.0

Exemplo
