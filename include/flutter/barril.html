Ao criar um aplicativo pronto para produção, é uma boa prática dividir os widgets em diferentes arquivos dart. Isso aumenta a legibilidade e ajuda na depuração, mas ao fazer isso, acabamos com toneladas de importações mais ou menos assim, como no exemplo abaixo: 
import 'package:flutter/material.dart';
import 'package:map_ui/screens/airplane.dart';
import 'package:map_ui/screens/bus.dart';
import 'package:map_ui/screens/train.dart';
import 'package:map_ui/screens/ship.dart';
Existe uma maneira de reduzir essas importações no início de cada arquivo, no flutter chama-se arquivos Barril. 
Um barril é uma maneira de acumular importações de vários arquivos em um único arquivo conveniente.
Para reduzir de várias importações para uma única importação, criamos um arquivo de barril.

Fazemos um barril criando um arquivo dart e colocando todas as importações nele.

criar arquivo transportes.dart

criar um arquivo com qualque nome, nesse exemplo transportes.dart

export arquivos

faca o export dos outros arquivos no arquivo de barril, transportes.dart 
export 'airplane.dart';
export 'bus.dart';
export 'ship.dart';
export 'train.dart';

substitua os imports

Substitua as quatro importações individuais importando o arquivo transportes.dart 
import 'package:map_ui/screens/transportes.dart';
