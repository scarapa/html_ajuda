Atualizar SDK/Dart

esse comando atualiza as depedencias de DESENVOLVIMENTO na maquina 
$ flutter upgrade

Novo Projeto

$ flutter create NOME_PROJETO

Baixar dependencias de projeto

Baixar as dependencias declaradas no arquivo pubspec.yaml do projeto. 
$ flutter pub get

Gerar APK

LOCAL

Essa tipo de release serve para testes, dificil de ser compartilhada entre devices por contas das politicas do SISTEMA OPERACIONAL. 
$ flutter run

HOMOLOGACAO

Essa release NÃO é possivel DEBUGAR 
$ flutter build apk
