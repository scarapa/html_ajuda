Gradle

Aqui estão as etapas completas para fazer o Gradle funcionar sem preocupações!.
  1. Baixe o Gradle em seu site: http://www.gradle.org/downloads
  2. Descompacte o pacote em qualquer lugar que você deseja mantê-lo. Eu normalmente coloco esse tipo de coisa no meu diretório de desenvolvimento, onde outras coisas também vão.
  3. Agora, clique com o botão direito em "Computador" (também conhecido como "Meu Computador") e selecione "Propriedades":
  4. Em seguida, clique em "Configurações avançadas do sistema":


  5. Agora, você já deve estar na guia "Avançado", então clique em "Variáveis ??de ambiente ...":


  6. Em seguida, queremos adicionar uma nova variável de usuário para nós, chamada JAVA_HOME. Clique em "Novo ..." conforme mostrado abaixo:


  7. Agora insira JAVA_HOME no campo de entrada para "Nome da variável:":


  8. Para "Valor da variável:", queremos inserir a localização do seu diretório Java JDK. O Java JDK em meu computador está localizado em seu local de instalação padrão, C:\ Arquivos de programas\Java\ jdk1.8.0_25


  9. Quando terminar, clique em "OK".
  10. Em "Variáveis ??do sistema", procure a "Variável" chamada Caminho


  11. Agora queremos adicionar caminhos adicionais à nossa variável de ambiente PATH, então vamos clicar em "Editar ..." em "Variáveis ??do sistema" conforme mostrado abaixo:


  12. Em seguida, prossiga para o final do valor e insira os locais \bin do Gradle e do Java JDK.


  13. Para Gradle, eu o inseri como C:\gradle-2.1\bin. Para o Java JDK, digite apenas %JAVA_HOME%\bin porque %JAVA_HOME% usará a variável de usuário que definimos anteriormente para obter o diretório Java JDK para você.


  14. Depois de inserir os novos caminhos de bin para o Gradle e o Java JDK, clique em "OK".
  15. Clique em "OK" mais algumas vezes para voltar atrás. Agora, vamos testar. Abra o prompt de comando e digite gradle -v e você verá algo semelhante:
