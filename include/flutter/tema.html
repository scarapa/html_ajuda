Dentro do MaterialApp seta-se uma das propriedades abaixo

Cores

primaryColor

Defina-se uma ÚNICA cor 
    theme: ThemeData(
        //primaryColor: Colors.purple,
      ),

primarySwatch

Essa cor faz parte do MateriColor, isso quer dizer que dentro de uma colorização pode ser usada um expectro de cores 
    theme: ThemeData(
        primarySwatch: Colors.purple[300],
      ),
E para utilizar o tema 
color: Theme.of(context).primaryColor,

accentColor

Propriedade de cor secundária 
return MaterialApp(
    home: homePage(),
    theme: ThemeData(
        primarySwatch: Colors.purple,
        accentColor: Colors.amber,
    ),
);

Fontes

Pasta

Necessário incluir na pasta desejada as fontes, no meu caso , foram dois tipos de Fontes
  • OpenSans
    • OpenSans-Regular
    • OpenSans-Bold
  • Quicksand
    • Quicksand-Regular
    • Quicksand-Bold

pubspec.yaml

Esse arquivo tem que ser muito bem identado, qualquer coisa errada acusa erro. 
fonts:
    - family: OpenSans
      fonts:
        - asset: assets/fonts/OpenSans-Regular.ttf
        - asset: assets/fonts/OpenSans-Bold.ttf
          weight: 700
    - family: Quicksand
      fonts:
        - asset: assets/fonts/Quicksand-Regular.ttf
        - asset: assets/fonts/Quicksand-Bold.ttf
          weight: 700
Utilizando diretamente 
    title: Text("Despesas",
    style: TextStyle(
      fontFamily: 'OpenSans'
    ),
),
Utilizando a partir do tema padrao ThemeData.light() existe varias opcoes dentro desse metodo
A propriedade title está depreceada, a propriedade corretá é headline6 conforme codigo abaixo: 
return MaterialApp(
    home: homePage(),
    theme: ThemeData(
        primarySwatch: Colors.purple,
        accentColor: Colors.amber,
        fontFamily: 'Quicksand',
        appBarTheme: AppBarTheme(
            textTheme: ThemeData.light().textTheme.copyWith(
            headline6: TextStyle(
                fontFamily: 'OpenSans',
                fontSize: 20,
                fontWeight: FontWeight.bold
                )
            )
        ),
    ),
);

style: Theme.of(context).textTheme.headline6,
