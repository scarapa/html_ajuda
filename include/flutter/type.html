const vs final

O Dart fornece palavras-chave const e final para criar variáveis ​​com valores fixos. No entanto, const é uma constante de tempo de compilação significa que seu valor deve ser declarado durante a compilação do programa e final é uma constante de tempo de execução, significa que seu valor pode ser atribuído durante a execução do programa.

Listas

O Dart fornece uma classe List que é usada para construir uma estrutura de dados do tipo Array que pode conter um número fixo ou variável de tipos de dados semelhantes ou dinâmicos. O Dart também fornece sintaxe literal [] para criar listas. 
    List<dynamic> fixedList = new List(3);
    List<num> fixedNumList = new List(3);
    List<String> fixedWordsList = new List('Teste 1');

    var myMap = new Map();

    //criando map com int como chave e a string como valor
    var fingers = newMap<int,String>();
    fingers[1] = 'thumb';
    fingers[2] = 'index';
Um mapa é mais como um dicionário em Python ou objeto em JavaScript. Basicamente, um mapa é uma coleção de pares chave: valor. O Dart fornece a classe Map para criar um mapa. Um mapa pode conter qualquer tipo de chave e valor. medium
