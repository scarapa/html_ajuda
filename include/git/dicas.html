Historico com os ultimos comandos executados

$ history | grep -i 'git'

ALIAS para criar comando de atalho

$ git config --global alias.st "status"

Para ver o arquivo que não foi commitado para o repositorio ORIGIN

$ git config --global alias.logdifforigin "log --name-only --pretty=format: master --not origin/master"
ou pesquisando pelo NOME DO SEU BRANCH 
$ git logdifforigin = git log --name-only --pretty=format: nome_do_seu_bracnh_local --not origin/master

Listar todos os arquivos commitados no dia

$ git log --since="1 day ago" --name-only --pretty=format: | sort | uniq
