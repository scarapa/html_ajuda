CLONE - 02

fatal: update_ref failed for ref 'HEAD': cannot lock ref 'HEAD': unable to resolve reference

SOLUCAO:
  1. criar o diretorio manualmente 
    $ sudo mkdir PROJETO_NOME
  2. startar git 
    $ git ini
  3. setar repositorio 
    $ git remote add origin PROJETO_REPOSITORIO
  4. buscar arquivos da branch padrao(master) 
    $ git pull
  5. largar esses arquivos na memoria local 
    $ git add .
    $ git commit -m "temp"
  6. listar branchs remotas 
    $ git branch -a
  7. buscar branch 
    $ git checkout -t remote/origin/PROJETO_BRANCH
CONCLUIDO !

PULL - 01

Erro completo abaixo: 
error: Your local changes to the following files would be overwritten by merge
*** LISTA DE ARQUIVOS ***
Please commit your changes or stash them before you merge.
Aborting
Acontece porque a sua versão é anterior a verão atual, portanto antes de fazer o commit e o push você precisa atualizar para que os conflitos sejam locais e nao no server.

Solução:
$ git checkout HEAD^ CAMINHO/DO/ARQUIVO
$ git pull
