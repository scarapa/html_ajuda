CLONE

PADRAO

clonar repositorio da forma tradicional 
$ git clone REPOSITORIO

PASSANDO USUARIO E SENHA

clonar repositorio passando usuario e senha; pode ser necessario caso o usuario GLOBAL na maquina nao seja o mesmo que tenha acesso ao repositorio. 
$ git clone https://USUARIO:SENHA@host.com/name/projeto.git
exemplo 
$ git clone https://SEU_USUARIO:SUA_SENHA@github.com/scarapa/html_ajuda.git

BRANCH

CRIAR Branch novo e ao mesmo tempo mudar para ele

$ git checkout -b nome_branch

RETORNAR para o Branch MASTER

$ git checkout master

REMOVER o Branch

$ git branch -D nome_branch

ALTERAR o nome do Branch

$ git branch -m nome_atual nome_novo

LISTAR Branchs

$ git branch

RESETAR branch

$ git checkout -f
ou 
$ git reset --HARD

PULL

PADRAO

$ git pull

PASSANDO USUARIO E SENHA

$ git pull https://user:password@github.com/name/repo.git BRANCH_PADRAO
exemplo 
$ git pull https://SEU_USUARIO:SUA_SENHA@github.com/scarapa/html_ajuda.git main
OBSERVACAO: no ano de 2021 o github alterou a branch padrao de MASTER para MAIN

COMMITs

Verifica o status dos arquivos

$ git status

Adicionar todos os arquivos ao commit

$ git add .

Adicionar apenas um arquivo ao commit

$ git add caminho_do_arquivo

Encapsula REALMENTE todos arquivos e adicionar a mensagem ao mesmo

$ git commit -m "mensagem"

Encapsula todos arquivos e adicionar a mensagem no MESMO COMANDO

$ git commit -am "mensagem"

NÃO incluir arquivo no COMMIT

NÃO incluir determinado arquivo que faz parte da sua configuração local, exemplo: retirar o arquivo editado por você da sua arvore de modificacoes , no nosso exemplo esse arquivo é de acesso de banco de dados; não tem necessidade de seu colega ou outra pessoa saber seu usuário e senha de banco.
$ git checkout NOME_ARQUIVO
Exemplo: 
$ git checkout Database.php

LOGs

Listar todos os arquivos commitados no dia

$ git log --since="1 day ago" --name-only --pretty=format: | sort | uniq
Fonte

Para ver o arquivo que não foi commitado para o repositorio ORIGIN

$ git config --global alias.logdifforigin "log --name-only --pretty=format: master --not origin/master"
ou pesquisando pelo NOME DO SEU BRANCH 
$ git logdifforigin = git log --name-only --pretty=format: nome_do_seu_bracnh_local --not origin/master
