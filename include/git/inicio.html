Caso JÁ TENHA o codigo fonte do projeto e quer simplesmente adicionar o GIT. 
$ git init
Criar o arquivo .gitignore, ele é responsável por dizer o que NUNCA vai ser comitado! 
$ touch .gitignore
Apontando para aonde está o seu repositorio 
$ git remote add origin URL/SSH
Adicionando os arquivos ao pacote 
$ git add .
"Fechando" o seu pacote e adicionando a mensagem para envio pro repositório 
$ git commit -m "Mensagem"
Enviando para o repositorio remoto! 
$ git push origin master
