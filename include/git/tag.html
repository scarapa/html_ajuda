TAG

Comandos Básicos

Verificar as tags existentes $ git tag
Criar nova tag $ git tag -a "v'numero_da_versao''" -m "mensagem informando o que foi feito nos commits que compõe a tag"
Verificar tag específica $ git tag -l "v1.8.5*"
Apresentar os commits que compõe a tag $ git show "v'numero_da_tag'"
