IE - Modo de Documento Emulado

Modo de documento emulado NÃO É a mesma coisa que modo de compatibilidade

Em algumas versões de IE, não existem determinados métodos javascript e muitos métodos jQuery, portanto alterar o modo de compatibilidade pode diminuir esses erros.

Em alguns casos, pode ser necessário restringir uma página da Web a um modo de documento suportado por uma versão mais antiga do Windows Internet Explorer. Aqui, vemos o cabeçalho compatível com x-ua-compatible, que permite que uma página da Web seja exibida como se fosse visualizada por uma versão anterior do navegador. 

<html>
	<head>
		<!-- Use Internet Explorer 9 Standards mode -->
		<meta http-equiv="x-ua-compatible" content="IE=9">
		<title>My webpage</title>
	</head>
	<body>
  		<p>Content goes here.</p>
	</body>
</html>
