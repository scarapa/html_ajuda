RETANGULOS

Diferente do SVG , o < canvas> suporta somente formas primitivas: retângulos. Todas as outras formas são criadas a partir da combinação de um ou mais caminhos (paths), lista de pontos conectados por uma linha. Felizmente, temos uma variedade de funções de desenho que tornam possíveis criar formas muito complexas. Primeiramente vamos olhar o retângulo. Aqui está listado três funções para desenhar retângulos pelo canvas:
function draw(){
	var canvas = document.getElementById('myCanvas');
	var ctx = canvas.getContext('2d');

	ctx.fillStyle = 'red';
	ctx.fillRect(10, 10, 60, 60);
	ctx.fillRect(100, 10, 100, 60);
}
Abaixo esta listado a função draw(), porém utilizando as três funções.
function draw() {
  var canvas = document.getElementById('myCanvas2');
  if (canvas.getContext) {
    var ctx = canvas.getContext('2d');

    ctx.fillRect(25, 25, 100, 100);
    ctx.clearRect(45, 45, 60, 60);
    ctx.strokeRect(50, 50, 50, 50);
  }
}
