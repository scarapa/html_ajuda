O objeto da janela representa uma janela aberta em um navegador.

Se um documento contém quadros (tags [iframe]), o navegador cria um objeto de janela para o documento HTML e um objeto de janela adicional para cada quadro.
closed Para criar objetos no banco de dados
defaultStatus Altera a estrutura da base de dados
document Remover todos os registros de uma tabela, incluindo todos os espaços alocados para os registros são removidos
frameElement Adicionar comentários ao dicionário de dados
frames Retorna todos os elementos [iframe] na janela atual
history Retorna o objeto Histórico para a janela (veja o objeto Histórico)
innerHeight Retorna a altura interna da área de conteúdo de uma janela
innerWidth Retorna a largura interna da área de conteúdo de uma janela
length Retorna o número de elementos [iframe] na janela atual
localStorage Permite salvar pares chave / valor em um navegador da Web. Armazena os dados sem data de validade
location Retorna o objeto Local para a janela (veja o objeto Local)
name Define ou retorna o nome de uma janela
navigator Retorna o objeto Navigator para a janela (veja o objeto Navegador)
opener Retorna uma referência para a janela que criou a janela
outerHeight Retorna a altura externa de uma janela, incluindo barras de ferramentas / barras de rolagem
outerWidth Retorna a largura externa de uma janela, incluindo barras de ferramentas / barras de rolagem
pageXOffset Retorna os pixels que o documento atual foi rolado (horizontalmente) do canto superior esquerdo da janela
pageYOffset Retorna os pixels que o documento atual foi rolado (verticalmente) do canto superior esquerdo da janela
parent Retorna a janela pai da janela atual
screen Retorna o objeto Tela para a janela (veja o objeto Tela)
screenLeft Retorna a coordenada horizontal da janela em relação à tela
screenTop Retorna a coordenada vertical da janela em relação à tela
screenX Retorna a coordenada horizontal da janela em relação à tela
screenY Retorna a coordenada vertical da janela em relação à tela
sessionStorage Permite salvar pares chave / valor em um navegador da Web. Armazena os dados para uma sessão
scrollX Um alias do pageXOffset
scrollY Um alias da páginaYOffset
self Retorna a janela atual
status Define ou retorna o texto na barra de status de uma janela
top Retorna a janela do navegador superior
