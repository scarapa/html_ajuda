O suporte de proxy é integrado ao Ionic CLI. As configurações de proxy podem ser definidas por meio do arquivo de configuração ou de uma variável de ambiente.

Para definir as configurações de proxy por meio do arquivo de configuração, execute o seguinte com a URL do servidor proxy:

INSTALACAO

Primeiramente instalar GLOBALMENTE 
$ npm install -g @ionic/cli-plugin-proxy
Depois necessário acessar a pasta do projeto e instalar o proxy no projeto 
$ cd myProject # cd into your project's directory
$ npm install --save-exact --save-dev @ionic/cli-plugin-proxy

CONFIGURACAO

Em seguida, use uma das seguintes variáveis ​​de ambiente: 
$ export HTTP_PROXY="http://proxy.example.com:8888" # also used by npm
$ export HTTPS_PROXY="https://proxy.example.com:8888" # also used by npm
$ export IONIC_HTTP_PROXY="http://proxy.example.com:8888"
