INCLUI

Possibilita verificar se um element existe ou não no array definido.
const beers = ['Heineken', 'San Diego', 'Coruja', 'Saint Bier'];
const ages = [20, 25, 19, 21, 42];
console.log(beers.includes('Skol')); // false > false console.log(ages.includes(25)); // true > true

FILTRAR

Este método possibilita a criação de filtros de forma rápida e fácil em seus arrays.
const beers = ['Heineken', 'San Diego', 'Coruja', 'Saint Bier'];
const ages = [20, 25, 19, 21, 42];
// ES5 function startWithS(word) { return word.indexOf('S') === 0; } // ES6 const startWithS = (word) => word.indexOf('S') === 0; const beersStartWithS = beers.filter(startWithS); console.log(beersStartWithS); // [0: 'San Diego', 1: Saint Bier] > [0: 'San Diego', 1: Saint Bier]

PROCURAR

Faça uma busca em seu array de um ou mais elementos que se encaixam no filtro definido por você.
const beers = ['Heineken', 'San Diego', 'Coruja', 'Saint Bier'];
const ages = [20, 25, 19, 21, 42];
// ES5 function findSanDiego(element) { return element === 'San Diego'; } // ES6 const findSanDiego = (element) => element === 'San Diego'; const beerSanDiego = beers.find(findSanDiego); console.log(beerSanDiego); // 'San Diego' > 'San Diego'

MAPEAR

Este método percorre todos os elementos do array, executando funções para cada elemento e retornando um novo array como resultado.
const beers = ['Heineken', 'San Diego', 'Coruja', 'Saint Bier'];
const ages = [20, 25, 19, 21, 42];
// ES5 function upAge(age) { return age + 1; } // ES6 const upAge = (age) => age + 1; const newAges = ages.map(upAge); console.log(newAges); // [0: 21, 1: 26, 2: 20, 3: 22, 4: 43] > [0: 21, 1: 26, 2: 20, 3: 22, 4: 43]

ALGUM

Este método verifica se ao menos um elemento satisfaz a condição definida por você.
const beers = ['Heineken', 'San Diego', 'Coruja', 'Saint Bier'];
const ages = [20, 25, 19, 21, 42];
// ES5 function hasHeinekenOrSaint(beer) { return (beer === 'Saint Bier' || beer === 'Heineken'); } // ES6 const hasHeinekenOrSaint = (beer) => (beer === 'Saint Bier' || beer === 'Heineken'); const heinekenSaint = beers.some(hasHeinekenOrSaint); console.log(heinekenSaint); // true > true

REDUZIR

Você pode utilizar o método reduce para alguns casos, sendo um deles para facilitar cálculos. .
const beers = ['Heineken', 'San Diego', 'Coruja', 'Saint Bier'];
const ages = [20, 25, 19, 21, 42];
// ES5 function reducerAge(accumulator, age) { return accumulator + age; } // ES6 const reducerAge = (accumulator, age) => accumulator + age; const sumAges = ages.reduce(reducerAge); console.log(sumAges); // 127 > 127
