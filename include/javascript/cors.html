CORS (Cross-Origin Resource Sharing) é uma especificação do W3C que, quando implementado pelo navegador, permite que um site acesse recursos de outro site mesmo estando em domínios diferentes.

Habilitando CORS no FIREFOX

1. Digite na URL

about:config
Abrirá uma janela de confirmação

2. Clicar em aceitar



Isso fará aparecer todas as configurações do navegador

3. Pesquisar pelo tipo origin



4. Localize o valor security.fileuri.strict_origin_policy


5. Altere o valor

Para alterar o valor para false, basta clicar 2x
