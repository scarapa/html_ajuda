TAG Script

async

Especifica que o script é baixado em paralelo para analisar a página e executado assim que estiver disponível (antes da conclusão da análise) (somente para scripts externos)

crossorigin

[anonymous/use-credentials]
Define o modo da solicitação para uma solicitação HTTP CORS

defer

[defer]
Especifica que o script é baixado em paralelo para analisar a página e executado depois que a página terminar de analisar (somente para scripts externos)

integrity

[filehash]
Permite que um navegador verifique o script obtido para garantir que o código nunca seja carregado se a fonte tiver sido manipulada

nomodule

[True/False]
Especifica que o script não deve ser executado em navegadores com suporte a módulos ES2015

referrerpolicy

[no-referrer/no-referrer-when-downgrade/origin/origin-when-cross-origin/same-origin/strict-origin/strict-origin-when-cross-origin/unsafe-url]
Especifica quais informações do referenciador enviar ao buscar um script

src

[url]
Especifica o URL de um arquivo de script externo

type

[scripttype]
Especifica o tipo de mídia do script

Manipulando a URL

Esse recurso é muito utilizado por aqueles sites que disponibilizam filmes p1r4ta5 online. Caso o usuário queira compartilhar a URL na rede social, o dono do site altera o URL para quando o usuário copiar a URL e postar, aparecer outro link e não a URL da página, evitando assim uma possível punição por direitos autorais .




Também é utilizado naqueles blogs que utilizam artigos enquanto rolar a página para baixo, ou seja, os artigos são carregados e automaticamente há alteração da URL caso o usuário queira compartilhar também .
