Toda vez que ocorre a interação com um documento web, um evento é disparado. Carregar ou fechar uma página caracteriza ocorrência de dois eventos distintos; em outras palavaras; evento é qualquer ocorrência interativa com um elemento de marcação.

Manipulador De Evento

É uma função a ser executada quando o evento é disparado.

Disparador De Evento

É o elemento HTML ao qual foi atrelado uma manipulador de evento.

Manipulador-padrão De Evento

O termo default event handler, que um tradução livre, significa manipulador padrao de evento, é uma funca padrao do navegador executada quando ocorrem determinados tipos de eventos.

Toda vez que se um manipulador de evento a um HTML, ele é executado antes do manipulador padrao

o DOM 2 preve os metodos addEventListener() e removeEventListener() destinados a atrelar e desatrelar respectivamente eventos a um nó do DOM.
