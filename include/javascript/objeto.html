Spread

permite que um objeto iterável, como uma expressão de array ou uma string seja expandido para ser usado onde zero ou mais argumentos (para chamadas de funções) ou elementos (para arrays literais) são esperados.

SEM spread

var middle = [3, 4];
var arr = [1, 2, middle, 5, 6];
console.log(arr);
// [1, 2, [3, 4], 5, 6]

COM spread

var middle = [3, 4];
var arr = [1, 2, ...middle, 5, 6];
console.log(arr);
// [1, 2, 3, 4, 5, 6]

COPY

Atualmente existem 3 formas de clonar um objeto; como exemplo utilizaremos o exemplo abaixo: 
let pessoa = { 'name': 'robert','last-name':'jordan','age':35  };

Object.assign

Alternativamente, Object.assign está no lançamento oficial e também criará uma cópia superficial do objeto. 
let pessoaCloneAssign = Object.assign({}, pessoa );
Observe o {} vazio como primeiro argumento, isso garantirá que você não altere o objeto original

Spread

Usar spread irá clonar seu objeto. Observe que esta será uma cópia superficial. A partir deste post, o operador spread para clonagem de objetos está no Estágio 4. 
let pessoaCloneSpread = { ...pessoa };

  
JSON

Esta forma final lhe dará uma cópia profunda. Agora vou mencionar que esta é uma maneira rápida e suja de clonar profundamente um objeto.

let pessoaCloneJson = JSON.parse(JSON.stringify(pessoa));
