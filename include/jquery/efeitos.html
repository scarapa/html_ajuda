# METODO DESCRICAO
animate()Executa uma animação personalizada nos elementos selecionados
clearQueue()Remove todas as funções enfileiradas restantes dos elementos selecionados
delay()Define um atraso para todas as funções na fila nos elementos selecionados
dequeue()Remove a próxima função da fila e, em seguida, executa a função
fadeIn()Desvanece-se nos elementos selecionados
fadeOut()Desvanece os elementos selecionados
fadeTo()Diminui / diminui os elementos selecionados para uma dada opacidade
fadeToggle()Alterna entre os métodos fadeIn () e fadeOut ()
finish()Pára, remove e conclui todas as animações em fila para os elementos selecionados
hide()Oculta os elementos selecionados
queue()Mostra as funções na fila nos elementos selecionados
show()Mostra os elementos selecionados
slideDown()Desliza para baixo (mostra) os elementos selecionados
slideToggle()Desliza para baixo (mostra) os elementos selecionados
slideUp()Desliza (oculta) os elementos selecionados
stop()Interrompe a animação atualmente em execução para os elementos selecionados
toggle()Alterna entre os métodos hide () e show ()
