Qual a diferença entre $(document).ready() e window.onload?
O evento ready é disparado depois que o documento HTML foi carregado.
O onload só é disparado quando todo o conteúdo é carregado (incluindo imagens, vídeos, etc).

Exemplo 1

$(document).ready(function() {
	//Começará o jQuery quando o DOM(Documento) estiver pronto
});

Exemplo 2

Igual ao exemplo 1 … literalmente a MESMA coisa. 
$(function(){ 

});

Exemplo 3

jQuery(document).ready(function($) {
	//Começará o jQuery quando o DOM(Documento) estiver pronto
});
A adição do jQuery pode ajudar a evitar conflitos com outros frameworks JS.
Os conflitos normalmente ocorrem porque muitas bibliotecas JavaScript / Frameworks usam o mesmo atalho Nome qual é o símbolo do dólar $. Então, se eles tiverem as mesmas funções mencionadas, o navegador obtém confuso!
Para evitar conflitos, recomendo aliasing o espaço de nome jQuery (ou seja, usando o exemplo 3 acima). Então, quando você chamar $ .noConflict () para evitar dificuldades de espaço para nome (como o atalho $ não está mais disponível) Nós estamos forçando-o a lidar com jQuery sempre que for necessário. 
jQuery.noConflict(); // Retorna a variável '$' de volta para outras bibliotecas JS
jQuery(document).ready( function(){ 
	//faça jQuery quando o DOM estiver pronto e sem conflitos
});  

//Ou a função de auto-execução
 jQuery.noConflict();
 (function($) { 
    // Código usando $ como alias para jQuery
})(jQuery);

Exemplo 4

Desta forma, você pode incorporar uma função dentro de uma função que usa o $ como um alias jQuery. 
(function($) { 
	//Código usando $ como alias para jQuery
  $(function() {
    //Mais código usando $ como alias para jQuery
  });
})(jQuery);
//Outro código usando $ como um alias para a outra biblioteca

Exemplo 5

$(window).load(function(){  
	//Inicializar depois que as imagens são carregadas
});
Às vezes você quer manipular imagens e com $ (documento) .ready () você não poderá fazer isso Se o visitante não tiver a imagem já carregada. Nesse caso, você precisa inicializar o Função de alinhamento jQuery quando a imagem termina de carregar.

Você também pode usar o JavaScript simples e anexar uma função chamar a tag do corpo no html, use isso somente se você não estiver usando uma estrutura JS.
