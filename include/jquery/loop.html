1. Basic jQuery.each()

Vamos ver como a função each() nos ajuda em conjunto com um objeto jQuery. O primeiro exemplo seleciona todos os elementos na página e exibe seu atributo href. 
$('a').each(function (index, value){
  console.log($(this).attr('href'));
});
O segundo exemplo emite cada href externo na página da Web (assumindo apenas o protocolo HTTP): 
$('a').each(function (index, value){
  var link = $(this).attr('href');

  if (link.indexOf('http://') === 0) {
    console.log(link);
  }
});
Digamos que na página tivemos os seguintes links:

Devemos notar que os elementos DOM de um objeto jQuery precisam ser embalados novamente quando usados ​​dentro de um jQuery each(). A razão é que jQuery é de fato apenas um invólucro em torno de uma série de elementos DOM. Ao usar jQuery each() esta matriz é iterada da mesma maneira que uma matriz comum seria. Portanto, não removemos elementos embrulhados da caixa.

2. jQuery.each() Array Example

var numbers = [1, 2, 3, 4, 5, 6];
$.each(numbers , function (index, value){
  console.log(index + ':' + value); 
});
This snippet outputs: 0:1, 1:2, 2:3, 3:4, 4:5, and 5:6. Nada especial aqui. Uma matriz possui índices numéricos, portanto, obtemos números a partir de 0 e indo até N-1, onde N é o número de elementos na matriz.

3. jQuery.each() JSON Example

Podemos ter estruturas de dados mais complicadas, como arrays em arrays, objetos em objetos, arrays em objetos ou objetos em matrizes. Vamos ver como each() pode nos ajudar em tais cenários. 
var json = [ 
 { 'red': '#f00' },
 { 'green': '#0f0' },
 { 'blue': '#00f' }
];

$.each(json, function () {
   $.each(this, function (name, value) {
      console.log(name + '=' + value);
   });
});
Este exemplo exibe red=#f00, green=#0f0, blue=#00f.

Nós lidamos com a estrutura aninhada com uma chamada aninhada para each (). A chamada externa lida com a matriz da variável JSON, a chamada interna lida com os objetos. Neste exemplo, cada objeto possui apenas uma chave, no entanto, em geral, qualquer número pode ser atacado com o código fornecido.

4. jQuery.each() Class Example

Este exemplo mostra como percorrer cada elemento com o produto da classe atribuído, dado no HTML abaixo. 
<div class="productDescription">Red</div>
<div>Pink</div>
<div class="productDescription">Orange</div>
<div class="generalDescription">Teal</div>
<div class="productDescription">Green</div>
Usamos each() ajudante em vez do método cada () no seletor. 
$.each($('.productDescription'), function (index, value) { 
  console.log(index + ':' + $(value).text()); 
});
Neste caso, a saída é 0:Red, 1:Orange, 2:Green.

Não precisamos incluir índice e valor. Estes são apenas parâmetros que ajudam a determinar qual elemento DOM que estamos atualmente iterando. Além disso, neste cenário, também podemos usar o método mais conveniente para cada um. Podemos escrever assim: 
$('.productDescription').each(function () { 
  console.log($(this).text());
});
E nós obteremos no console: 
Red
Orange
Green
Mais uma vez, precisamos envolver o elemento DOM em uma nova instância jQuery. Usamos o método text () para obter o texto do elemento para saída.

5. jQuery .each() Delay Example

No próximo exemplo, quando o usuário clicar no elemento com o ID 5demo, todos os itens da lista serão ajustados para laranja imediatamente. Após um atraso dependente do índice (0, 200, 400, ... milissegundos), desaparecemos o elemento. 
$('#5demo').bind('click', function (e) {
  $('li').each(function (index) {
    $(this).css('background-color', 'orange')
		.delay(index * 200)
		.fadeOut(1500);
  });
  e.preventDefault();
});
