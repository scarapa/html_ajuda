Principais eventos JQUERY. Geralmente os eventos serão utilizados dessa maneira, exemplo do evento click abaixo:
$( "#target" ).click(function() {
});

.bind()

'Ouvindo' o elemento

.blur()

Utilizado quando o foco/ponteiro deixa o elemento

.change()

Quando há alteração no valor ou status do elemento

.click()/.dblclick()

.contextmenu()

.focus()/.focusin()/.focusin()

Serve para setar o ponteiro no elemento ou quando o focu entra/sai do elemento.

.hasClass()

O método .hasClass () retornará true se a classe estiver atribuída a um elemento, mesmo que outras classes também o sejam. Por exemplo, dado o JS abaixo 
let st_active = $("#menu li").hasClass("active")

.keydown()/.keypress()/.keyup()

.off()/.on()
