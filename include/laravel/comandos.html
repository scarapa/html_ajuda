MAKE

O make possibilita a criação de código fonte, tornando assim o desenvolvimento altamente produtivo.Alguns exemplos:
cria uma nova classe controller 
$ php artisan make:controller
cria uma nova classe model; 
$ php artisan make:model
cria um novo arquivo de migração; 
$ php artisan make:migration

MIGRATIONS/BANCO

O comando abaixo, executa as operações no banco de dados, criando tabelas, rodando script sql entre outros. 
$ php artisan migrate
dropar tudo o que foi feito, resetando ao estado original; 
$ php artisan migrate:fresh

Rotas

O route possibilita a criação de rotas, tornando a aplicação mais segura.Alguns exemplos:
cria um arquivo de rota cache de registro rápido; 
$ php artisan route:cache
remove o arquivo de rota do cache 
$ php artisan route:clear
apresenta todas as rotas registradas na aplicação 
$ php artisan route:list

Aplicacao

Caso sua aplicação esteja em produção e você precise realizar alguma manutenção que a aplicação necessite ficar alguns minutos fora

Derrubar

O comando abaixo, fará com que sua aplicação apresenta a tela manutenção padrão do Laravel e não fique off-line. 
$ php artisan down

Levantar

Assim que a equipe de desenvolvimento concluir a manutenção basta apenas digitar o seguinte comando no prompt de comando: 
$ php artisan up

Variaveis de Ambiente

Apresenta o ambiente de desenvolvimento que está sendo utilizado. 
$ php artisan env

Servidor

Local

$ php artisan serve

Remoto

php artisan serve --host 192.168.20.20 --port 8000
