Introdução

Conceito


Pasta

app\Http\Controllers

Criando

$ php artisan make:controller MeuControllador
Criando controler com metodos padrões 
$ php artisan make:controller NomeControlador --resource

Requisicoes HTTP

Criando metodo que automatiza todos os metodos , exemplo o controller de clientes 
Route::resource('clientes','ClienteControlador');
Exibindo todas rotas: 
$ php artisan route:list

Associando ROTAS ao CONTROLADOR

Metodos

Index

mostrar todos os elementos

Create

criar um NOVO elemento

Store

salvar o novo elemento

Show

exibir determinado elemento

Edit

editar determinado elemento

Update

alterar de terminado elemento JA EXISTENTE

Destroy

deletar elemento
