Criar projeto

Criando um projeto com o nome de extranet com a ultima versao do laravel! 
$ composer create-project --prefer-dist laravel/laravel extranet
OBSERVAÇÃO
Atualmente novembro/2019 o laravel está na versão 6.0, como proceder caso deseja-se instalar a versão 5.8?
Para isso basta executar o comando dessa forma: 
composer create-project --prefer-dist laravel/laravel NOME_DO_PROJETO "5.8.*"

Server

comando para levantar servido HTTP
