046 - Criação de um database em Linux

criar database 
mysql> create database laravel_produtos
exibir databases 
mysql> show database
acessar database 
mysql> use laravel_produtos
exibir tabelas 
mysql> show tables
exibir COLUNAS da tabela products 
mysql> describe products
Comando para criar usuario com todas permissoes 
$ sudo mysql 
mysql > GRANT ALL PRIVILEGES ON *.* TO 'user'@'localhost' IDENTIFIED BY 'user';

048 - Criando migrations

$ php artisan make:migration create_feira
Aplicando as migracoes 
$ php artisan migrate

Adicionando um campo a uma tabela

php artisan make:migration add_marca_to_products --table=products
