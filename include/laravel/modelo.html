055 - Criando Modelos

Criar modelo:
$ php artisan make:model NOME-CLASSE

056 - Inserindo registros

Utiliza-se o tinker 
php artisan tinker;
'incluir' o model antes' 
App\NOME-DA-CLASSE
1º forma de SALVAR 
//1º Instanciando a classe		
use App\Feiras\Categoria
$categoria = new Categoria

//2º Setando o objeto
$categoria->descricao="Mel"
$categoria->cd_matricula=49583
$categoria->dt_criacao="2019-03-09 14:35:12.000"

$categoria->save()
2º forma de SALVAR 
//1º Instanciando a classe		
use App\Feiras\Categoria
$categoria = Categoria::create(["descricao"=>"mel","cd_matricula"=>49583,"dt_criacao"=>"2019-03-09 14:35:12.000"])
