1º ir para diretorio raiz

$ cd

2º criar o arquivo .bashrc

$ touch .bashrc

3º editar o arquivo .bashrc, no windows podemos utilizar o editor notepad

WINDOWS
$ notepad .bashrc
LINUX
$ nano .bashrc

4º exemplos de aliases

alias sisema_nome="cd w:/intranet/sistemas/sisema_nome/"
alias gpm="git push origin master"

5º executar/efetivar as alterações

$ source .bashrc
