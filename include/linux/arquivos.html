Acrescentar texto em arquivo

simplesmente adicionar os dados ao arquivo sem apagar os dados existentes.

operador >>

O operador >> redireciona a saída para um arquivo. Se o arquivo mencionado não existir, o arquivo é criado e o texto é anexado ao arquivo. 
$ echo "Which is the best Linux Distro?"  >>  file.txt
Alternativamente, podemos usar o comando printf para anexar texto a um arquivo. 
$ printf "Which is the best Linux Distro?\n"  >>  file.txt
OBSERVACAO
Não use > de >>, pois isso apagará os dados do arquivo de destino. Isso pode causar perda de dados.

JUNTANTO ARQUIVOS

Também podemos usar o comando cat para anexar o conteúdo de um arquivo a outro arquivo. 
$ cat file1.txt >> file2.txt
