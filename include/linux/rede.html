Setando IP manualmente

Verifique qual o nome do seu arquivo de configuração de rede com: 
$ ls /etc/netplan/
retorno: 01-network-manager-all.yaml 
$ sudo nano /etc/netplan/50-cloud-init.yaml
exemplo de como ficou o meu arquivo, detalhe , é um arquivo do tipo .yaml , ou seja , a identação conta em muitooooo. 
    # Let NetworkManager manage all devices on this system
    network:
      version: 2
      renderer: NetworkManager
      ethernets:
        enps8s0:
          dhcp4: no
          addresses: [192.168.8.21/24]
          gateway4: 192.168.8.1
          nameservers:
           addresses: [8.8.8.8, 8.8.4.4]
teste 
$ sudo netplan try
aplique 
$ sudo netplan apply
Verifique 
$ ip a
