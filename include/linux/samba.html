DOWNLOAD

Baixando a aplicação SAMBA do repositorio oficial

UBUNTU 14

$ sudo apt-get install samba samba-common python-glade2 system-config-samba

UBUNTU 20

$ sudo apt-get install -y

BACKUP

Antes de editar o arquivo de configuração fazer uma cópia por segurança.

UBUNTU 14

$ sudo cp /etc/samba/smb.cnf /etc/samba/smb.cnf.bak

UBUNTU 20

$ sudo cp /etc/samba/smb.conf /etc/samba/smb.cnf.bak

CONFIGURACAO

UBUNTU 14

$ sudo nano /etc/samba/smb.cnf

UBUNTU 20

$ sudo nano /etc/samba/smb.conf
Inserir configuracao abaixo no fim do arquivo: 
[NOME_DO _COMPARTILHAMENTO]
path = /CAMINHO/DA/PASTA
browsable =yes
writable = yes
guest ok = yes
read only = no

REINICIALIZAR SERVICO

$ sudo service smbd restart 
//ou 
$ sudo /etc/init.d/smbd restart
