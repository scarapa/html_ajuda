NOVO PROJETO

criar nova pasta do projeto novo 
$ mkdir NOVO_PROJETO
acessar a pasta 
$ cd NOVO_PROJETO
iniciar o projeto node 
$ npm init
com a opcao sim/yes para TODAS perguntas 
$ npm init -y

LISTAR PACOTE

LOCAL

Se você quiser ver uma lista estruturada em árvore de todos os seus pacotes instalados localmente, incluindo suas dependências 
$ npm list

GLOBAL

Se você quiser verificar os pacotes instalados globalmente, basta adicionar o sinalizador -g (abreviação de -global): 
$ npm list -g

PROFUNDIDADE

Claro, você pode simplesmente acessar o arquivo package-lock.json, mas a verificação manual das versões dos pacotes envolverá alguma verificação visual. Se você deseja exibir todos os pacotes de nível superior instalados sem suas dependências, adicione a flag -depth=0. A flag é usada para limitar a profundidade da árvore de dependência que pode ser exibida. 
$ npm list -depth=0

PESQUISAR PACOTE

PESQUISAR PACOTE INSTALADO

Se você quiser saber a versão mais recente específica de um pacote instalado, basta especificar seu nome. 
$ npm list [package-name]

PESQUISAR VERSAO MAIS RECENTE

Se você deseja verificar a versão mais recente de um pacote disponível no repositório npm, pode usar o comando npm view package-name version. 
$ npm view express version
  4.18.2
