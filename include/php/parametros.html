VERBOS

Para pegar os verbos GET ou POST no php 
$_SERVER['REQUEST_METHOD']

GET

Geralmente para pegar as variaveis enviadas pelo GET, fariamos da seguinte forma 
// enviando via URL
?nome=fulano&sobrenome=silva

// recebendo
$nome = $_GET['nome'];
$sobrenome = $_GET['sobrenome'];

POST

// formulario
nome:<input type="text" name="nome" value="fulado"><br>
sobrenome:<input type="text" name="sobrenome" value="silva"><br>    

// recebendo
$nome = $_POST['nome'];
$sobrenome = $_POST['sobrenome'];
