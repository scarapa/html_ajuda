RAILS

Comandos Básicos

Checar versão $ rails -v
Ajuda $ rails -h
OU
$ rails --help
Abrir console $ rails console
OU
$ rails c
Iniciar o servidor (Puma) interno do rails $ rails server
OU
$ rails s
-e para selecionar um ambiente diferente
-p para selecionar uma porta diferente
$ rails s -e production -p 4000
Iniciar um novo projeto $ rails new app_name
Gerar um novo controller para o projeto
É necessário colocar um nome para o controller. Opcionalmente é possível já criar o controller com as actions.
$ rails generate controller nome_controller [action action] [options]
Exemplo:
$ rails generate controller Greetings hello
Gerar um novo model para o projeto
É necessário colocar um nome para o model. Opcionalmente é possível criar o model com os campos farão parte da sua composição, colocando seu nome e tipo.
$ rails generate model NAME [field[:type][:index]]
Tipos disponíveis no momento
  • integer
  • primary_key
  • decimal
  • float
  • boolean
  • binary
  • string
  • text
  • date
  • time
  • datetime
  • timestamp
Exemplo:
$ rails generate model Article title:string text:text
Exemplo criado o campo user como string e o campo pseudo como string e unique:
$ rails g model user pseudo:string:uniq
Fazer uma migração $ rails db:migrate
Rollback de migração $ rails db:rollback
Executar os seeds $ rails db:seed
