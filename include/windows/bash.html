BASH LINUX NO WINDOWS

Como instalar e usar o Shell Bash do Linux no Windows 10

Isto é possível por causa do chamado “Subsistema do Windows para o Linux”, que a Microsoft está adicionando ao Windows 10 e é resultado de uma parceria da empresa com a Canonical. Se você quer experimentar essa novidade, veja neste tutorial como instalar e usar o Shell Bash do Linux no Windows 10, e comece a explorá-la.

Acesse as configurações do Windows 10. Para isso, clique no menu Iniciar e depois na opção “Configurações”;


Em “Configurações”, desça a tela até a o opção “Atualização e segurança” e clique nela;




marque o item “Modo de desenvolvedor”


clique no botao SIM


marque a opcao SUBSISTEMA DO WINDOWS PARA LINUX

DESABILITE O HYPER-V

Na janela da linha de comando, você precisa executar o comando bcdedit /set hypervisorlaunchtype off para desabilitar o Hyper-V em seu sistema. bcdedit é uma ferramenta de linha de comando para gerenciar armazenamentos de dados de configuração de inicialização (BCD). Você pode conferir mais sobre isso no site oficial da Microsoft. 
C:\>bcdedit /set hypervisorlaunchtype off
    The operation completed successfully.

DESATIVE RECURSO DO WINDOWS

Em seguida, você precisa desativar o recurso "Virtual Machine Platform" nos Recursos do Windows. Para isso, basta desmarcar esse recurso e clicar em OK.

REINICIE

Depois de desmarcar o recurso Virtual Machine Platform, ele solicitará que você reinicie o Windows para atualizar as alterações. Clique em reiniciar agora para reiniciar o sistema Windows. Isso é tudo que você precisa fazer. Depois de concluir as etapas, agora você pode tentar iniciar a VM novamente e, desta vez, deve iniciar sem nenhum erro.
